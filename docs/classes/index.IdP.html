exchangeCode

  • exchangeCode(code: string, client: string, clientSecret: string, redirectUri: string): Promise<AccessToken>

  • Once the client has obtained an authorization code, they will attempt to exchange it for an access token.

    This function takes an authorization code (grant) and returns an access token. It revokes the authorization code (so it can only be used once) and stores the access token in the database.

verifyAccessToken

  • verifyAccessToken(token: string, user: string, scope: string): Promise<AccessToken>

  • When the API is queried, requests will include an access token (e.g. in an Auhorization header with a value of "Bearer ...")

    This function checks that the access token is valid for the provided user and scope.

    The scope is optional, and if not provided, the function will check that the access token is valid for any scope. If provided, scope is verified by checking that the access token's scope .includes() the provided scope.

    Returns

    The Access token

    Parameters

    • token: string

      The token to check

    • user: string

      The ID of the user which is making the request

    • scope: string

      The scope of the request

    Returns Promise<AccessToken>
